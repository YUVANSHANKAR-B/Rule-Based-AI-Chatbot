# Portfolio Summary

## Project: Rule-Based AI Chatbot

### What it is
A beginner-friendly chatbot built with Python that uses keyword-based rules to answer user questions.

### Why it matters
- Demonstrates basic AI/chatbot design without machine learning
- Shows ability to structure code into reusable modules
- Includes both a command-line interface and a web interface

### Key skills showcased
- Python programming
- conditional logic and rule matching
- use of Flask for a simple web application
- project documentation and testing

### How to present this in an internship application
- "Built a rule-based chatbot project in Python with a CLI and a Flask web UI."
- "Designed keyword-driven conversation rules, implemented modular code, and added test coverage."
- "Prepared clear documentation and a project summary to explain how the system works."
